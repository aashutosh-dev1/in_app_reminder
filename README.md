# in_app_reminder

Flutter verson: 3.29.2

A Flutter plugin for adding native reminders to the iOS Reminders app using `EventKit`.

- ✅ Set title and notes
- ✅ Add date and time
- ✅ Set daily, weekly, monthly, or yearly repeat
- ❌ Android support: coming soon

---

## ✨ Features

| Feature             | Supported |
|---------------------|-----------|
| iOS Reminders       | ✅        |
| Repeat options      | ✅        |
| Time-based alarm    | ✅        |
| Android support     | ❌ Coming soon |
